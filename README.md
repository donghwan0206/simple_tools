# Simple Alias Switcher for Elasticsearch

## Install & Run
### 1. Manual
#### Prerequisite
- python 3.11.9(or later)
- pip3

```shell
git clone https://github.com/donghwan0206/simple_tools.git
cd simple_tools
pip3 install -r requirements.txt
stremlit run src/index.py 
```


### 2. Using docker
#### Prerequisite
- docker engine

```shell
git clone https://github.com/donghwan0206/simple_tools.git
cd simple_tools
docker compose up
```
